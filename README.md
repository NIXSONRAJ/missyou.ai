# missyou.ai ✨
A premium, AMOLED-themed Computer Science Mentor powered by AI.

## Features
- **CS Mentorship:** Powered by Gemini 2.0 Flash (via OpenRouter/Any OpenAI API).
- **Document Brain:** Upload PDFs or code files for context-aware learning.
- **AMOLED Interface:** Minimalist, high-contrast design for deep focus.
- **Interactive:** Features Copy, Regenerate, and Text-to-Speech toggles.
- **Portable:** Can be compiled into a standalone Windows `.exe`.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python missyou.py`
3. Enter your API key in the onboarding screen.

## Technologies
- Python / Flask
- LangChain / FAISS (RAG)
- HTML5 / CSS3 / Vanilla JS
